## pyenv, poetry

### pyenv 란
> **pyenv**는 여러 Python 버전을 관리할 수 있게 해주는 도구이다  
시스템에 여러 파이썬 버전을 설치하고 프로젝트에 따라 설치한 파이썬 버전을 쉽게 전환할 수 있다

### poetry 란
> **Poetry**는 파이썬 프로젝트의 의존성을 관리하고 패키지를 빌드 및 배포하는데 도움을 주는 도구이다  
기존에는 `venv`를 통해 가상환경을 구성하고 pip를 통해 라이브러리를 설치하고 이러한 번거로움이 있었지만 `Poetry`는 `venv`와 통합되어 가상 환경을 자동으로 생성하고 관리할 수 있다  

#### pyproject.toml
- 파이썬 프로젝트의 빌드 시스템과 메타 데이터를 정의하고 프로젝트의 의존성을 관리하는 파일.

### 가상환경 구축

[패키지 버전 충돌 관리]
- 여러 프로젝트가 서로 다른 패키지를 필요로 할 수 있기 때문에 가상환경을 사용하여 서로 독립적인 환경으로 관리한다   

`venv`,`conda`등 다양한 도구를 통해 가상환경을 생성하고 관리할 수 있다

## Python 튜플(Tuple)
- 튜플은 리스트와 비슷하지만 차이점이 있다, 튜플은 불변 객체로 생성 후에 절대 값이 변경될 수 없다
- 튜플은 요소가 삽입된 순서를 기억한다(리스트와 동일)  


튜플 언패킹
```python
# 튜플 생성
person = ("Alice", 30, "Engineer")

# 튜플 언패킹
name, age, profession = person
```
변수의 수와 요소의 수가 맞지 않으면 에러 발생
